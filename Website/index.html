🤖 Agentic Architecture

Multi-Agent Verification Pipeline

Five specialized AI agents collaborate like a team of experts. Each agent brings unique capabilities, and their outputs converge into a consensus verdict.

News Article / URL Input

User submits content for verification

Agent 1

Claim Extraction Agent

Parses article → identifies individual factual claims → passes claim list downstream

Claims sent in parallel to three specialist agents
Agent 2

Evidence Agent

Fetches supporting & contradicting sources from trusted databases

Agent 3

Credibility Agent

Scores each source domain by trust history & editorial reliability

Agent 4

Bias & Emotion Agent

Detects fear language, clickbait, urgency cues & manipulation

All outputs converge into consensus agent
Agent 5

Consensus & Verdict Agent

Weighs evidence + source credibility + bias signals → Confidence-calibrated final verdict
